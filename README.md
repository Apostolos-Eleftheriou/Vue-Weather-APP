This project showcases a weather application built with VueJS and Tailwind, leveraging the Open Meteo API for weather forecasts and OpenStreetMap for location services. Users can search and select locations, view detailed hourly and daily weather forecasts — including temperature, wind speed, and direction — and save their favorite locations locally for quick access even after page reloads. The application features extensive data manipulation to combine and display information from both APIs properly and effectively to the user.
